# 感情分析キーロガーアプリケーション

キーロガーから感情分析への変換を行う統合GUIアプリケーションです。

## 機能

- **キーロガー機能**: リアルタイムでキー入力を記録
- **感情分析機能**: HMMモデルを使用した感情推定
- **リアルタイム分析**: 記録中にリアルタイムで感情を推定
- **データ保存**: CSV形式でのデータ出力
- **タブ式インターフェース**: キーログ、感情分析結果、リアルタイム分析を分離表示

## 必要な依存関係

```bash
pip install -r requirements.txt
```

## 使用方法

1. **アプリケーションの起動**
   ```bash
   python emotion_keylogger_app.py
   ```

2. **キーロガーの開始**
   - 「キーロガー開始」ボタンをクリック
   - キー入力を開始すると自動的に記録が開始されます

3. **感情分析の実行**
   - 十分なデータが記録されたら「感情分析実行」ボタンをクリック
   - HMMモデルが読み込まれている場合、感情分析結果が表示されます

4. **データの保存**
   - 「CSV保存」ボタンでキーログデータをCSVファイルに保存

## タブ説明

- **キーログ**: 記録されたキー入力の詳細情報
- **感情分析結果**: 実行した感情分析の結果
- **リアルタイム分析**: リアルタイムでの感情推定結果

## HMMモデルについて

アプリケーションは事前に訓練されたHMMモデル（`hmm_emotion_model_4states.pkl`）を使用します。
モデルが存在しない場合は、先にモデルの訓練が必要です。

## 感情ラベル

- 感覚運動的興奮
- 難解・頭脳型
- 和みと癒し
- 設定状況の魅力

## 注意事項

- キーロガーは機密情報を記録する可能性があります。使用時は注意してください
- HMMモデルが存在しない場合、感情分析機能は利用できません
- リアルタイム分析は最近のキー入力データに基づいて実行されます
